# Baltic Sea indicator time series

Time series of two marine indicators covering the period 1984–2016 in
the Eastern Baltic Sea (data altered from original time series). This
dataset serves as a demo input in the
[`model_sensitivity`](https://helenegutte.github.io/ecorisk/reference/model_sensitivity.md)
and
[`model_exposure`](https://helenegutte.github.io/ecorisk/reference/model_exposure.md)
functions.

## Usage

``` r
indicator_ts_baltic
```

## Format

A data frame with 33 observations and 3 variables.

- year:

  Time variable.

- zooplankton_mean_size:

  Mean size of zooplankton (in wet weight micrograms).

- eastern_baltic_cod:

  Mean spawning stock biomass of eastern Baltic cod (units unspecified).
